  DOS Coreutils includes some parts of GNU Coreutils.

## Usage

    command [Arguments]
    command --help|--version

Argument --help is for help.

Argument --version is for version information.

## Available Commands

  * beep
  * cat
  * cd
  * clear
  * cp
  * date
  * dirname
  * echo
  * false
  * help
  * ls
  * mkdir
  * mv
  * pwd
  * rm
  * rmdir
  * sh
  * time
  * true

## Compile

This software should be compiled with Open Watcom.
